# Requirements: Sidebar menu

The following document describes the requirements to create a sidebar menu for the application.

ID: HP-06

## Description

Create a sidebar menu for use throughout the application.

## Acceptance Criteria

- The sidebar component should be added from the shadcn/ui Sidebar component.
- The sidebar may be toggled open and closed using a sidebar trigger button located in the application header.
- The sidebar header should display the application name.
- The sidebar should contain a collapsible sidebar group for the "Documentation" page family.

shadcn/ui sidebar: https://ui.shadcn.com/docs/components/sidebar
